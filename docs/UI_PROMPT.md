# AI 코치 웹앱 — UI 생성 마스터 프롬프트

> **용도**: 이 프롬프트를 Claude Code, Cursor, v0.dev 등 AI 코딩 도구에 그대로 붙여넣어 Next.js 기반 UI를 한 번에 생성한다.

---

## 📋 프로젝트 컨텍스트

너는 시니어 UX 전문가이자 Next.js 풀스택 엔지니어다.

다음 조건의 웹앱 UI 전체를 구현해라:

- **서비스명**: LearnBridge (AI 리스킬링 코치)
- **핵심 가치**: AI 시대에 뒤처지기 쉬운 중장년 직장인이 5분 안에 자신의 학습 경로를 받고, 매주 성장을 확인할 수 있는 AI 코칭 웹앱
- **기술 스택**: Next.js 14 (App Router) + TypeScript + Tailwind CSS + shadcn/ui
- **주 타깃**: 40~60대 직장인 (디지털 친숙도 보통~낮음)
- **핵심 UX 원칙**: 큰 글씨, 높은 대비, 단순한 네비게이션, 단계별 안내, 실수해도 안전한 흐름

- **카드 패딩**: 24px
- **라운드**: rounded-2xl (큼직하게)

## 📱 구현할 페이지 목록

### 1. 랜딩/로그인 페이지 `/`

**레이아웃**: 전체 화면 2-컬럼 (좌: 브랜드 영역, 우: 로그인 폼)

**좌측 패널** (bg-gradient: Primary→#1E40AF):
- 로고 (LearnBridge + 작은 뇌+경로 아이콘)
- 헤드라인: "AI가 설계하는\n나만의 성장 경로"
- 서브: "5분 진단으로 시작하는 맞춤 학습 코칭"
- 사회적 증거 3줄: 아이콘 + 한 줄 강점 (예: ✓ 매주 새로운 학습 계획 / ✓ 중장년 전용 UI / ✓ AI 코치 24시간 대기)

**우측 패널** (흰 배경):
- "시작하기" h1 제목
- 이메일 + 비밀번호 입력 (큰 레이블, placeholder 포함)
- 로그인 버튼 (Primary, full-width, 52px)
- 구분선 "또는"
- 구글 로그인 버튼 (outline, full-width)
- "처음이신가요? 무료로 시작하기" 링크

**반응형**: 모바일에서 단일 컬럼, 로그인 폼만 표시

### 2. 온보딩 페이지 `/onboarding`

**컨셉**: 5단계 스텝 위저드, 한 화면에 하나의 질문만

**공통 레이아웃**:
- 상단: 진행바 (0~100%) + "2/5 단계" 텍스트
- 중앙: 질문 카드 (최대 width 640px, 중앙 정렬)
- 하단: 이전 / 다음 버튼

**Step 1 — 직업/역할 선택**:
- 제목: "현재 어떤 일을 하고 계신가요?"
- 선택지 카드 6개 (아이콘 + 텍스트, 탭으로 선택):
  - 💼 직장인 (회사원/공무원)
  - 🏢 자영업자/프리랜서
  - 🔍 취업준비 중
  - 📚 재교육/이직 준비
  - 🏠 경력 단절 후 복직 준비
  - 🌱 기타

**Step 2 — 학습 목표**:
- 제목: "AI를 통해 가장 이루고 싶은 것은 무엇인가요?"
- 선택지 카드 (다중 선택 가능, 최대 3개):
  - 📊 업무 생산성 향상
  - 💡 새로운 기술 습득
  - 🔄 직무 전환/이직
  - 🤖 AI 도구 활용 능숙해지기
  - 📈 승진/경력 개발
  - 🛡️ 고용 안정성 확보

**Step 3 — 현재 AI 활용 수준 (진단)**:
- 제목: "현재 AI 도구를 얼마나 사용하고 계신가요?"
- 대형 슬라이더 (1~5단계):
  - 1: "ChatGPT가 뭔지 잘 모릅니다"
  - 3: "가끔 써봤지만 잘 모릅니다"
  - 5: "업무에 적극 활용하고 있습니다"
- 선택한 단계 설명 텍스트 (실시간 변경)

**Step 4 — 주당 학습 가능 시간**:
- 제목: "일주일에 학습에 쓸 수 있는 시간은?"
- 카드 선택 4개:
  - ⏰ 1시간 미만
  - 🕑 1~3시간
  - 🕓 3~5시간
  - 🕗 5시간 이상

**Step 5 — 온보딩 완료**:
- 체크 애니메이션 (에메랄드 원 + 체크 아이콘)
- 제목: "준비 완료! 나만의 학습 경로를 생성 중입니다"
- AI 처리 중 애니메이션 (3개 점이 순차적으로 깜빡임)
- 완료 후 대시보드로 자동 이동

### 3. 메인 대시보드 `/dashboard`

**레이아웃**: 사이드바 + 메인 콘텐츠 영역

**사이드바** (width: 260px, bg: Neutral-900):
- 상단: 유저 아바타 + 이름 + "안녕하세요 👋"
- 메뉴 항목 (active 시 Primary 배경):
  - 🏠 홈 (대시보드)
  - 📋 이번 주 학습 계획
  - 📈 진행 현황
  - 💬 AI 코치에게 질문
  - ⚙️ 설정
- 하단: 로그아웃 버튼

**메인 콘텐츠**:

**상단 헤더 카드** (Primary 그라디언트):
- "이번 주 목표" 제목
- 진행률 링 차트 (중앙: 퍼센트 숫자)
- 오른쪽: 완료 과제 N/전체 N

**3열 요약 카드** (각각):
- 📅 이번 주 학습 일수 (숫자 크게 + "일")
- ⏱️ 누적 학습 시간 (숫자 + "시간")
- 🔥 연속 학습 (숫자 + "주 연속")

**이번 주 학습 계획 섹션**:
- 제목 + "전체 보기 →" 링크
- 학습 아이템 카드 3~4개 (체크박스 + 제목 + 예상 시간 + 난이도 배지)
- 각 카드: 클릭 시 체크 완료 애니메이션

**AI 코치 추천 메시지**:
- 왼쪽: 로봇/AI 아바타 아이콘 (Primary 원형 배경)
- 오른쪽: 개인화된 격려 메시지 카드 (말풍선 스타일)
- 예: "김태진님, 지난주보다 학습 시간이 20% 늘었어요! 이번 주엔 'AI 프롬프트 기초' 완료를 목표로 해봐요."

### 4. 주간 학습 계획 페이지 `/plan`

**레이아웃**: 전체 너비, 상단 주차 네비게이션

**상단**:
- "< 이전 주 | 2026년 4월 1주차 | 다음 주 >" 네비게이션
- AI 재생성 버튼 (outline, 오른쪽 상단)

**계획 카드 목록**:

각 학습 항목은 카드 형태:
```
┌─────────────────────────────────────────────┐
│ [체크박스]  📖 ChatGPT 프롬프트 기초          │
│             난이도: ●●○○○ 초급              │
│             예상 시간: 45분                   │
│             [시작하기 →]  [AI에게 질문]       │
└─────────────────────────────────────────────┘
```
- 완료된 항목: 배경 에메랄드 틴트 + 취소선
- 항목별 "왜 이 학습이 필요한가" 펼치기 버튼

**주간 계획 생성 근거 섹션** (접을 수 있는 아코디언):
- "AI가 이 계획을 추천한 이유"
- 진단 결과 요약 → 목표 → 계획 연결 설명

### 5. 진행 현황 페이지 `/progress`

**레이아웃**: 대시보드와 동일한 사이드바

**상단 요약 카드 4개** (수평 배치):
- 총 학습 주차
- 완료한 학습 항목 수
- 총 학습 시간
- AI 레벨 향상 (입문 → 초급 → 중급)

**주간 진행 타임라인**:
- 세로 타임라인 형태
- 각 주: 날짜 + 완료 항목 + 성취 배지 + 짧은 AI 피드백
- 상태 아이콘:
  - ✅ 완료 주 (에메랄드)
  - 🔄 진행 중 주 (Primary)
  - ⚠️ 미완료 주 (앰버)

**역량 레이더 차트** (recharts):
- 축: AI 이해도 / 도구 활용 / 문제 해결 / 협업 / 자기주도학습
- 2개 라인: 시작 시점 vs 현재

**월별 학습 히트맵** (깃허브 스타일):
- 각 셀: 날짜별 학습 여부
- 색: 흰색 → 에메랄드 (연하게 → 진하게)

### 6. 주간 피드백 체크인 페이지 `/checkin`

**컨셉**: 간단한 설문 형식, 3~4개 질문만

**헤더**:
- "이번 주 어떠셨나요?" 대형 텍스트
- 현재 주차 표시

**질문 1 — 전반적인 학습 경험**:
- 이모지 선택 (😣 / 😐 / 🙂 / 😊 / 🤩)
- 선택 시 배경색 변화 + 간단한 반응 텍스트

**질문 2 — 완료한 학습 항목 체크**:
- 이번 주 계획 항목 리스트 + 체크박스

**질문 3 — 어려웠던 점 (선택)**:
- 텍스트 에어리어 (optional, placeholder: "AI 코치에게 자유롭게 남겨주세요")

**제출 버튼** + 완료 후 축하 애니메이션:
- 화면 중앙에 색종이(confetti) 파티클
- "이번 주도 고생하셨어요! AI 코치가 다음 주 계획을 업데이트했습니다." 메시지

### 7. AI 코치 질문 페이지 `/coach`

**레이아웃**: 채팅 인터페이스 스타일

**상단 바**:
- AI 코치 아바타 + "LearnBridge AI 코치"
- 온라인 상태 표시 (초록 점)

**채팅 영역**:
- 메시지 버블 (AI: 왼쪽 Primary 배경 / 사용자: 오른쪽 Neutral 배경)

- 타이핑 중 애니메이션 (점 3개)

**입력 영역**:
- 큰 텍스트 에어리어 (placeholder: "궁금한 것을 편하게 물어보세요")
- 전송 버튼 (Primary, 아이콘 only, 원형)
- 빠른 질문 버튼 3개:
  - "이번 주 계획 설명해줘"
  - "학습 속도가 맞나요?"
  - "다음 단계는 뭔가요?"

## 🚫 금지 사항 (Must NOT)

- 복잡한 드롭다운 메뉴 중첩 금지
- 10px 이하 텍스트 사용 금지
- 모달 위에 모달 금지
- 페이지 로드 없이 갑작스러운 콘텐츠 변화 (CLS) 금지
- 색상만으로 정보 구분하는 UI 금지 (아이콘/텍스트 병행 필수)
- 자동 재생 미디어 금지

---

## ✅ 접근성 필수 항목

- 모든 인터랙티브 요소에 aria-label 또는 시각적 텍스트
- 키보드 탭 순서가 논리적으로 연결
- 포커스 링 명확히 표시 (outline: 3px Primary)
- 색 대비 WCAG AA 기준 이상 (일반 텍스트 4.5:1, 대형 텍스트 3:1)
- 200% 확대 시 핵심 UI 요소 사라지지 않음

## 💡 특별 지시사항

이 앱은 **중장년 직장인**을 위한 앱이다. 다음을 항상 염두에 두어라:

- 전문 용어는 쉬운 말로 병기한다 (예: "AI 프롬프트 = AI에게 명령하는 방법")
- 에러 메시지는 친근하고 해결책 중심으로 작성한다
- "잘못된 입력입니다" ❌ → "이메일 형식을 다시 확인해 주세요 😊" ✅
- 진행 상황을 자주 보여줘서 사용자가 방향을 잃지 않도록 한다
- 성취 순간마다 작은 축하 요소(애니메이션, 배지)를 넣어 동기부여를 높인다
