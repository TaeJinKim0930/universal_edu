# 공모전 제출 체크리스트

## 1. 저장소
- [ ] GitHub 저장소가 public이다.
- [ ] `.env`, API Key, 서비스 계정 키가 노출되지 않았다.
- [ ] README에 프로젝트 설명이 있다.

## 2. 배포
- [ ] 라이브 URL이 정상 접속된다.
- [ ] 핵심 데모 플로우가 실제로 동작한다.
- [ ] 모바일/데스크톱 기본 레이아웃이 깨지지 않는다.

## 3. AI 리포트
- [ ] 문제정의가 명확하다.
- [ ] 왜 AI가 필요한지 설명된다.
- [ ] AI와 협업한 과정이 드러난다.
- [ ] 기획/검증/개선 흐름이 기록되어 있다.

## 4. 심사기준 대응
- [ ] 기술적 완성도를 보여주는 증빙이 있다.
- [ ] AI 활용 능력/효율성을 설명할 수 있다.
- [ ] 기획력과 실무 접합성이 드러난다.
- [ ] 창의성 포인트가 한 문장으로 설명된다.

## 5. 서류
- [ ] 개인정보 수집/이용 동의서 준비 완료
- [ ] 참가 각서 준비 완료
- [ ] 팀원 서명 여부 확인 완료

## 6. 마감 리스크
- [ ] 마감 이후 커밋이 발생하지 않도록 체크했다.
- [ ] 최종 제출 파일 목록을 재확인했다.
- [ ] 대표 제출자 기준 전달물 정리 완료
